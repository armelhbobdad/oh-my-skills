# Oh-My-Skills

Curated collection of [agentskills.io](https://agentskills.io)-compliant, version-pinned agent skills — generated and maintained by [Skill Forge](https://github.com/armelhbobdad/bmad-module-skill-forge).

## What is this?

Agent skills are structured documentation packages that give AI coding assistants (Claude Code, Cursor, Copilot) accurate, version-specific knowledge about libraries and frameworks. Instead of relying on stale training data, agents read skill files at runtime for up-to-date API signatures, types, and usage patterns.

Each skill in this repository is:

- **AST-verified** — extracted from source code with provenance citations, not hallucinated
- **Version-pinned** — tied to a specific library version, so agents know exactly what API exists
- **Two-tier** — inline quick reference + progressive disclosure for deep dives
- **Passively injected** — context snippets in `CLAUDE.md` tell agents when to load a skill

## Usage

### Install a skill from this repository

```bash
npx skills add https://github.com/armelhbobdad/oh-my-skills/tree/main/skills/<skill-name>
```

This installs the skill into your project's directory. Then you should manually add the context snippet from `skills/<skill-name>/context-snippet.md` to your `CLAUDE.md` (or `.cursorrules` / `AGENTS.md`).

### With Claude Code

Once installed, skills are available via the managed section in `CLAUDE.md`. When you write code that uses a listed library, Claude reads the corresponding `SKILL.md` before responding.

### Publish a skill

```bash
npx skills publish <skill-name>
```

Only `official` skills (generated by the library maintainer) should be published to the [agentskills.io](https://agentskills.io) ecosystem. Community skills like those in this repo are for local use.

## How skills are made

Skills are compiled using [Skill Forge](https://github.com/armelhbobdad/bmad-module-skill-forge) ([docs](https://armelhbobdad.github.io/bmad-module-skill-forge/)), which:

1. **Briefs** the skill scope through guided discovery
2. **Extracts** exports via AST analysis (ast-grep) with ephemeral sparse-checkout clones
3. **Enriches** with temporal context (GitHub issues, PRs, releases) indexed into QMD
4. **Compiles** into agentskills.io-compliant SKILL.md with provenance citations
5. **Tests** for completeness (coverage, coherence, external validation)
6. **Exports** with passive context injection into CLAUDE.md/AGENTS.md/.cursorrules

## Repository structure

```
skills/              # Skill packages (agentskills.io-compliant)
  <name>/
    SKILL.md           # Main skill document
    context-snippet.md # Compressed context for CLAUDE.md
    metadata.json      # Machine-readable metadata
    references/        # Progressive disclosure detail files

forge-data/          # Compilation workspace (provenance maps, evidence reports)
_bmad/skf/           # Skill Forge framework
CLAUDE.md            # Managed context injection for Claude Code
```
